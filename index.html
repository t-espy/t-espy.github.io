lean-optimizer — Parallel Optimization Engine for Algorithmic Trading

Production-grade parallel optimization engine for QuantConnect LEAN algorithmic trading strategies. Compiles a C# strategy once, spins up a pool of warm Docker containers with persistent .NET harness processes, and searches the parameter space using an incremental genetic algorithm. Additional pipeline stages (LHS, Bayesian/Optuna TPE, Local Grid) are available and configurable. Measured head-to-head against LEAN CLI's built-in grid search on a 679,140-point parameter space: the GA converged in 611 evaluations (1,112x fewer) at 1.8x higher throughput per worker — completing in under 3 minutes what grid search would take ~88 hours. 177 passing tests. Running on a DGX Spark (20-core, 128GB).

langgraph-factory — Multi-Agent Code Generation with Local LLMs

Reference implementation of a multi-agent code generation pipeline using LangGraph and locally-hosted LLMs via Docker Model Runner. Demonstrates how to decompose complex, multi-file code generation into discrete agent nodes — a foreman model (gpt-oss 20B) handles architecture planning and code review, while a coder model (Qwen3 Coder 80B) handles file generation and fixes — with an automated build-fix-regenerate loop that recovers from failures without human intervention. Includes mechanical error recovery (deterministic fixes without LLM calls), spin detection, and failure-informed regeneration. Produces buildable Next.js App Router projects from a natural-language spec. Runs entirely on local hardware (DGX Spark, 128GB unified RAM) with no external API dependencies. Includes a cost and data privacy analysis comparing local inference vs. per-token cloud APIs.

Route Forge — SaaS Multi-Stop Route Planning

End-to-end design, development, and deployment of Route Forge (route-forge.com), a SaaS web application for multi-stop route planning. Built on Next.js, React, Tailwind, Supabase PostgreSQL, and Stripe, hosted on Vercel with automated GitHub Actions CI/CD. Architected as a three-app system — planner, storefront, and API/auth — using the Backend for Frontend (BFF) pattern with cross-subdomain authentication for security and scalability. Demonstrates full production deployment from zero: payments, auth, SEO setup, and go-to-market infrastructure. Built using AI coding assistants throughout — GitHub Copilot, Amazon Q Developer, and OpenAI Codex — demonstrating AI-accelerated full-stack development from architecture through deployment.
