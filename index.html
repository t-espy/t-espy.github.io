lean-optimizer — Parallel Optimization Engine for Algorithmic Trading

Production-grade parallel optimization engine for QuantConnect LEAN algorithmic trading strategies. Compiles a C# strategy once, spins up a pool of warm Docker containers with persistent .NET harness processes, and searches the parameter space using an incremental genetic algorithm. Additional pipeline stages (LHS, Bayesian/Optuna TPE, Local Grid) are available and configurable. Measured head-to-head against LEAN CLI's built-in grid search on a 679,140-point parameter space: the GA converged in 611 evaluations (1,112x fewer) at 1.8x higher throughput per worker — completing in under 3 minutes what grid search would take ~88 hours. 177 passing tests. Running on a DGX Spark (20-core, 128GB).

Route Forge — SaaS Multi-Stop Route Planning

End-to-end design, development, and deployment of Route Forge (route-forge.com), a SaaS web application for multi-stop route planning. Built on Next.js, React, Tailwind, Supabase PostgreSQL, and Stripe, hosted on Vercel with automated GitHub Actions CI/CD. Architected as a three-app system — planner, storefront, and API/auth — using the Backend for Frontend (BFF) pattern with cross-subdomain authentication for security and scalability. Demonstrates full production deployment from zero: payments, auth, SEO setup, and go-to-market infrastructure.

Local Agentic Code Generation — LangGraph + Docker Model Runner

Local agentic code generation pipeline built on LangGraph, orchestrating DeepSeek R1 and Qwen3 Coder Next served via Docker Model Runner on a DGX Spark (20-core, 128GB). The pipeline uses DeepSeek R1 for reasoning and planning, Qwen3 Coder for code generation, and LangGraph for agent orchestration — producing deployable web application scaffolding end-to-end without external API dependencies. Demonstrates GPU-accelerated local LLM inference as a development force multiplier: the same hardware stack and containerization approach powers the lean-optimizer parallel optimization engine.
